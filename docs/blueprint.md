# **App Name**: NEU Library Visitor Log

## Core Features:

- Visitor Sign-in Portal: An intuitive interface for visitors to sign in using their ID or institutional email, along with their purpose of visit.
- Admin Login & Authentication: Secure administrative access to view and manage visitor logs and settings.
- Real-time Visitor Statistics Dashboard: A dashboard displaying up-to-date metrics such as today's visitors, weekly visitors, blocked individuals, and active sessions.
- Detailed Visitor Activity Log: A searchable and filterable table providing a comprehensive record of all visitor entries, including time in, name, college/office, and purpose.
- Block List Management: An administrative tool to view, add, or remove users from a block list, detailing their status and potential reasons for blocking.
- AI-powered Purpose of Visit Tool: An intelligent tool that suggests common purposes of visit to streamline the sign-in process for visitors.
- PDF Report Generation: Enables administrators to generate and export visitor statistics and activity logs into a printable PDF format for record-keeping.

## Style Guidelines:

- Primary color: A deep, professional forest green (#14663A), chosen for its association with institutions, stability, and knowledge. This color anchors the identity and provides a sense of established reliability, prominent in headers and key UI elements.
- Background color: A subtle, desaturated light green-grey (#EFF5F2), providing a clean and calming canvas for content while maintaining a hint of the primary brand hue. It ensures high readability for extensive textual content like logs and reports.
- Accent color: A vibrant, clear green (#33CC33), analogous to the primary but much brighter, used for 'Active' status indicators, calls to action, and interactive elements to draw attention effectively and convey a sense of progress or positive action.
- Informational color: A clear and impactful red (#D94D4D) specifically for 'Blocked' statuses and alerts, providing immediate visual feedback for critical information, maintaining consistency with common UX patterns for warnings.
- All text: 'Inter' (sans-serif) for its modern, legible, and neutral aesthetic, ensuring excellent readability across both headlines and body text, fitting the functional and informative nature of a library system.
- Clean, line-based icons for clarity and modernity. Specific icons for administrative login (person), search (magnifying glass), report generation (document/PDF), and status indicators (checkmarks, alert signs) will be consistent in style.
- A responsive two-column layout on larger screens, dividing visitor input from admin functions. Information display leverages card-based elements for statistics and structured tables for logs and block lists, emphasizing clear hierarchy and data presentation.
- Subtle and functional animations, such as smooth transitions for data updates in tables, understated hover effects on buttons and interactive elements, and brief loading indicators, to enhance user experience without distraction.
